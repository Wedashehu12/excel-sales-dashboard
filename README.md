# excel-sales-dashboard
Interactive Excel dashboard analyzing sales performance by category and region using PivotTables, PivotCharts, KPIs, and slicers.
## Project Overview
This project is an interactive Excel dashboard built using retail sales data. The dashboard analyzes sales performance across categories and regions using PivotTables, PivotCharts, KPIs, and slicers.

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- KPI Reporting

## Dashboard Features
- Total Sales KPI
- Sales by Category
- Sales by Region
- Interactive Region Slicer
- Dashboard Layout

## Skills Demonstrated
- Data analysis
- Business reporting
- Dashboard design
- Excel analytics
- Interactive filtering
